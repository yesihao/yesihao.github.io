前言

Ahmad Shadeed 的文章css-object-view-box写得很好，他可以让我们对object-view-box属性的有一定的了解，他将其描述为在浏览器中使用 CSS 裁剪图像的原生方式。

如何使用

使用案例呢？好吧，Ahmad 开头展示了如何在没有css-object-view-box的情况下来完成一些图片裁剪需求。
